# Match Intelligence v5 — Central de Respostas

Esta versão muda o foco do sistema: ele deixa de apenas exibir números e passa a responder perguntas.

## Novidades
- campo “Pergunte ao banco”
- respostas em linguagem natural
- evidências utilizadas na resposta
- análise automática do jogador
- melhor e pior set
- vitórias e derrotas por número do set
- análise de dias da semana e horários
- primeiro set, confirmação e poder de virada
- histórico completo
- comparador H2H

O campo de perguntas usa um interpretador local baseado nas informações registradas, sem depender de API externa.
